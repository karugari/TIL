# Day 01: HTML Introduction

## 1. 웹을 구성하는 3대 언어
1. **HTML(구조):** 웹사이트의 내용과 구조를 정의.
2. **CSS (스타일):** 웹사이트의 디자인과 스타일을 정의.
3. **JavaScript(기능):** 웹사이트의 동적인 상호작용

## 2. 핵심용어

- **URL:** 웹사이트 주소. 방문하고 싶은 곳에 대한 요청을 보냄.
- **HTTP/HTTPS:** 웹 서버와 통신하기 위한 표준 프로토콜(규약). `S`는 Secure(보안)를 의미하며 데이터가 암호화됨.
- **Domain (도메인):** `google.com` 처럼 사람이 기억하기 쉽게 만든 웹사이트의 식별자.
- **IP Address:** `192.168.0.1` 처럼 인터넷상의 모든 기기가 가진 고유 주소. 컴퓨터는 도메인이 아닌 IP 주소로 서로를 찾음.
- **DNS (Domain Name System):** 도메인 이름(google.com)을 IP 주소로 변환해주는 '인터넷 전화번호부' 역할.
- **Server (서버):** 웹사이트의 파일(HTML, CSS 등)을 저장하고 있다가, 요청이 오면 보내주는 컴퓨터.
    - *개발 중에는 내 컴퓨터(Localhost)가 임시 서버 역할을 함.*
